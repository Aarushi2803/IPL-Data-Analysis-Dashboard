# 🏏 IPL Data Analysis Dashboard

###  One‑line tagline
Uncovering IPL trends: wins, toss decisions, venues, and star performers.

---

## 📖 Project Overview
This project explores **Indian Premier League (IPL) data since 2008** to answer real analytical questions.  
It demonstrates skills in **data cleaning, visualization, and storytelling** using Excel, with charts that make cricket insights easy to understand.  

---

## ⚙️ Tech Stack
- **Excel ** → Pivot tables, slicers, charts ,Interactive dashboards
- **Data Sources** → IPL match & player datasets  

---

## 🔍 Analysis Highlights
- **Matches won by team wrt Bat First vs Field First (2008–present)**  
- **Toss decision based winner % (batting vs fielding choice)**  
- **Top 10 venues with most matches and win distribution (Bat First vs Field First)**  
- **Top 10 Man of the Match award winners**  
- **Championship titles won by teams**  

---

## 📊 Features
- Interactive **slicers** to filter by team, season, or venue  
- Clean **pivot tables** for quick summaries  
- Visualizations: bar charts, line graphs, proportional rectangles  
- Insights written in plain language for non‑technical readers
